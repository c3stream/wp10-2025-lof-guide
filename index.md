# WP10 2025 対戦企画攻略ガイド

> **免責事項**: このガイドは有志による非公式情報であり、**未検証の内容を含みます**。実際のゲームプレイで異なる結果になる可能性があります。

**LOCK ON FLEEK GAME FESTA ルール対応版**

10年間で凱旋門賞と国内GIを制覇できる最強馬を作るための完全攻略ガイドです。

## 対戦ルール

| 項目 | 設定 |
|------|------|
| 開始年 | 2012年 |
| 難易度 | ノーマル |
| スペシャル種牡馬/繁殖牝馬 | なし |
| 結婚 | なし |
| 育成期間 | 10年まで |
| 史実馬の育成 | なし（1年目譲渡馬は除く） |
| DLCアイテム | なし |
| **出走登録** | **1頭のみ・3歳馬限定** |
| **ピーク設定** | **あり** |
| 騎手 | 万能君 |
| 対戦レース | 凱旋門賞 + 国内GI抽選3レース |

**抽選対象レース**: 安田記念 / 宝塚記念 / 日本ダービー / 天皇賞秋 / 菊花賞

> ⚠️ **重要**: 対戦馬は**3歳馬限定**です。10年目（2021年）の3歳馬 = **2018年生まれ**の馬を育成する必要があります。

## 勝利への5つの鍵

1. **ウィンドエスプリで金策** - 晩成・持続で長期活躍、スタミナ82（金策専用、対戦馬ではない）
2. **仔出し10の繁殖牝馬を確保** - クリソプレーズ、ラヴズオンリーミー
3. **爆発力40以上の配合** - 母系インブリード活用で最強3歳馬を生産
4. **欧州牧場開設** - 凱旋門賞対策に必須
5. **2018年生まれの3歳馬を育成** - 成長型「早め」で3歳ピークを狙う

## 対戦馬の目標スペック（3歳馬）

| 項目 | 最低ライン | 理想値 |
|------|----------|--------|
| スピード | 75 (A) | 78+ (S) |
| スタミナ | 60 | 70+ |
| 距離適性 | 1800-2600m | 1600-3000m |
| 柔軟性 | B+ | A+ |
| **成長型** | **早め** | **早め・持続** |
| **成長力** | **ある** | **持続** |

> ⚠️ **注意**: 3歳馬限定のため、成長型「遅め」「晩成」は避ける。「早め」で3歳時にピークを迎える馬を狙う。
