## 施設建設優先順位

### 最優先（1年目）

| 施設 | 効果 | 備考 |
|------|------|------|
| **ミスト** | 調子・気合乗り維持 | 面積消費なし |
| **トレッドミル** | 疲労軽減 | 面積消費なし |
| **観光施設Lv1** | 安定収入（月1000万） | 7月から建設可能 |
| **研究棟Lv2** | 牧場長成長促進 | 早期建設推奨 |

### 高優先（2-3年目）

| 施設 | 目標Lv | 効果 |
|------|--------|------|
| 獣医施設 | Lv3×2 | 幼駒の怪我防止 |
| ウォーキングマシン | Lv3×2 | 疲労回復＋受胎率UP |
| **種牡馬繋養施設** | **Lv3** | **爆発力+3、海外種牡馬種付け可能** |

### 中優先（4年目以降）

| 施設 | 効果 |
|------|------|
| 温泉施設Lv3×2 | 競走寿命減少緩和 |
| ロンギ場Lv1 | 牧野双葉解放条件 |
| 厩舎 | 繁殖牝馬頭数増加 |
| 花壇Lv1〜2 | 幼駒のサブパラUP |

### 欧州牧場（凱旋門賞対策に必須）

| 条件 | 達成目安 |
|------|----------|
| 名声5000以上 | 3年目 |
| 生産者リーディング10位以内 | 3年目 |

## 逆算スケジュール（重要）

対戦馬を2022年3歳で完成させるための逆算:

```
2022年 → 対戦馬3歳（登録・出走）
2021年 → 対戦馬2歳（育成）
2020年 → 対戦馬誕生（0歳）
2019年 → 対戦馬の親を種付け
2018年 → 親となる馬の能力確定・因子獲得
2017年 → 親となる馬の祖父母世代完成
```

**8年目（2019年）の種付けが勝負の分かれ目。**
