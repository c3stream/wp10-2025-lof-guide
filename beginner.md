## 馬の見方（基本パラメータ）

### メインパラメータ

| パラメータ | 意味 | 重要度 |
|-----------|------|--------|
| **スピード (SP)** | 馬の最高速度、勝敗を分ける最重要値 | ★★★★★ |
| **スタミナ (ST)** | 持久力、長距離レースで重要 | ★★★★ |

### サブパラメータ

| パラメータ | 意味 | 重要度 |
|-----------|------|--------|
| 瞬発力 | 最後の直線での加速力 | ★★★★ |
| 勝負根性 | 競り合いでの強さ | ★★★★ |
| パワー | 坂や重馬場での強さ | ★★★ |
| 柔軟性 | 距離適性の幅 | ★★★★ |
| 精神力 | 海外遠征・大レースでの力発揮 | ★★★ |

### 能力値の目安

| ランク | 数値目安 | 評価 |
|--------|----------|------|
| S | 80以上 | 超一流 |
| A | 70-79 | 一流 |
| B | 60-69 | 重賞級 |
| C | 50-59 | オープン級 |
| D | 40-49 | 条件級 |

## 幼駒コメントの読み方

牧場長が幼駒を見た時のコメントで、おおよその能力がわかります。

### スピード判定コメント

| コメント | SP推定値 | 評価 |
|----------|----------|------|
| **「とんでもない素質」** | 75以上 | 最強クラス |
| **「かなりのもの」** | 70〜74 | GI級 |
| 「なかなかのもの」 | 65〜69 | 重賞級 |
| 「特別な雰囲気がない」 | 64以下 | 売却候補 |

### サブパラ判定（美香の印）

| 印 | 意味 |
|----|------|
| ◎ | 最も優れている |
| ○ | 優れている |
| △ | 普通 |
| 無印 | やや劣る |

**ポイント**: 「とんでもない素質」コメントの幼駒は必ず残しましょう。

## 成長型の理解

馬は成長型によって能力の伸び方が異なります。

### 成長型一覧

| 成長型 | 特徴 | 対戦向き |
|--------|------|----------|
| 早熟 | 2-3歳で能力ピーク、早く衰える | △ |
| 早め | 3-4歳でピーク | ○ |
| 遅め | 4-5歳でピーク | ◎ |
| 晩成 | 5歳以降にピーク | ○ |
| 超晩成 | 6歳以降にピーク | △ |

### 成長力

| 成長力 | 意味 |
|--------|------|
| **持続** | 長期間能力を維持 |
| 普通 | 標準的な成長カーブ |
| あり | 成長余地あり |
| なし | すぐにピークを迎える |

**対戦向け推奨**: 「遅め・持続」または「早め・持続」

## 調教の基本

### 調教方針

| 方針 | 効果 | 使いどころ |
|------|------|------------|
| **スピード重視** | SP上昇しやすい | 能力強化期 |
| **距離適性重視** | 距離幅拡大 | 万能馬育成 |
| バランス重視 | サブパラ全体UP | 総合力強化 |

### 気合乗りの重要性

| 気合乗り | レース結果への影響 |
|----------|------------------|
| 絶好調 | 能力を最大限発揮 |
| 好調 | 良い結果が期待できる |
| 普通 | 標準的な力を発揮 |
| イマイチ | 実力を出し切れない |
| 絶不調 | 大敗の危険 |

**ポイント**: 重要レース前は「絶好調」か「好調」を目指しましょう。

### 史実調教（推奨）

| 調教名 | 効果 |
|--------|------|
| 坂路セパレート追い | パワー・柔軟性UP |
| 坂路調教のさきがけ | 一時的に距離適性UP |
| ウッドチップ追い | 調子UP |

## レースの基本

### 脚質（走り方）

| 脚質 | 特徴 | 必要サブパラ |
|------|------|------------|
| **逃げ** | 先頭を走り続ける | 勝負根性、スタミナ |
| **先行** | 前の方でレースを進める | 勝負根性、パワー |
| **差し** | 後方から追い上げる | 瞬発力、パワー |
| **追込** | 最後方から一気に追い込む | 瞬発力、精神力 |

### 距離適性

| 距離カテゴリ | 距離 | 代表レース |
|-------------|------|-----------|
| スプリント | 〜1400m | 高松宮記念 |
| マイル | 1400-1800m | 安田記念 |
| 中距離 | 1800-2200m | 天皇賞秋 |
| 中長距離 | 2200-2600m | 日本ダービー、凱旋門賞 |
| 長距離 | 2600m〜 | 菊花賞、天皇賞春 |

### 重要なレース選び

- **適性距離内のレースを選ぶ**（距離適性外は大幅不利）
- **馬場適性を確認**（芝/ダート）
- **疲労を溜めすぎない**（連戦は避ける）

## 強い幼駒を出すための基礎

### 繁殖牝馬の選び方

| 項目 | 重要度 | 説明 |
|------|--------|------|
| **仔出し** | ★★★★★ | 10が最高、産駒の能力上限に直結 |
| 能力 | ★★★ | SP・サブパラが高いほど良い |
| 因子 | ★★★★ | SP因子持ちが理想 |

### 種牡馬の選び方

| 項目 | 重要度 | 説明 |
|------|--------|------|
| **SP** | ★★★★★ | 75以上が理想 |
| 因子 | ★★★★ | SP因子2個持ちが最良 |
| 配合相性 | ★★★★ | 爆発力に直結 |

### 爆発力とは

配合時に表示される「爆発力」が高いほど、優秀な産駒が生まれやすくなります。

| 爆発力 | 期待できる産駒 |
|--------|---------------|
| 20未満 | 国内重賞級 |
| 20-30 | 国内GI級 |
| 30-40 | 海外GI級 |
| **40以上** | **世界最強級** |

**対戦目標**: 爆発力40以上の配合を目指しましょう。
