## 対戦ルール概要

- 開始年: 2012年
- 難易度: ノーマル
- 史実馬: 1年目譲渡馬のみ使用可
- 育成期間: 10年まで
- 対戦レース: 凱旋門賞 + 国内GI抽選3レース
