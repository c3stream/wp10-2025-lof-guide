# ウイニングポスト10 2025 対戦企画向け完全攻略ガイド

**1年目譲渡馬のみを使用する10年間育成ルールで凱旋門賞と国内GⅠを制覇できる馬を作る方法を詳解する。** 2012年スタート時に入手できる架空馬の中から高スピード馬を厳選し、爆発力40以上の配合を10年間で構築することが勝利への鍵となる。本ガイドでは、初期繁殖牝馬の選定から配合理論の最適化、年度別行動チャートまで、対戦ルールに完全対応した攻略情報を網羅する。

## 爆発力40以上を達成する配合理論

対戦用の最強馬生産には**爆発力40以上**が目標となる。以下の配合理論を組み合わせて到達する。

### 爆発力加算の主要要素

| 配合理論 | 爆発力 | 達成条件 |
|----------|--------|----------|
| 種牡馬施設Lv3 | +3 | 名声1000以上で建設可能 |
| 血脈活性化（8本） | +8 | 2代前4頭の親系統を3-4種類に分散 |
| フォースニックス | +8 | 4系統のニックス関係成立 |
| 母父◎（因子2個） | +4 | 系統確立済み＋因子2個の種牡馬を母父に |
| 活力源化大種牡馬因子 | +10〜 | 父母両方の先祖に大種牡馬因子 |
| 真稲妻/真疾風配合 | +12 | 毛色条件＋瞬発/根性条件 |
| 母系インブリード | +8/本 | SP因子2個持ちで母系インブリード |
| **合計例** | **40+** | 上記を複数組み合わせ |

### 稲妻配合・疾風配合の条件

**稲妻配合**（爆発力+6）
- 父母ともに瞬発力**A以上**、勝負根性**D+以下**
- 効果：瞬発力・柔軟性UP、勝負根性DOWN

**真稲妻配合**（爆発力+12）
- 稲妻配合成立＋両親ともに**芦毛か白毛**

**疾風配合**（爆発力+6）
- 父母ともに勝負根性**A以上**、瞬発力**D+以下**
- 効果：勝負根性・精神力UP、瞬発力DOWN

**真疾風配合**（爆発力+12）
- 疾風配合成立＋両親ともに**青毛・青鹿毛・黒鹿毛**

### 母系インブリードの活用法

ウイポ10の新要素である母系インブリードは通常インブリードより**効果が高く危険度が低い**。

- **効果**：スピード因子1個につき爆発力**+4**（通常は+3）
- **危険度**：通常より**1低い**
- **条件**：父の母系と母の母系（4代以内）に共通の馬

**ミスタープロスペクターやダンジグ**など**SP因子2個持ち**の馬で母系インブリードを2本成立させると、爆発力+16〜+18が狙える。

### 架空馬のみで構築するニックス戦略

10年間でニックス関係を意図的に構築するには、**系統確立**が必要。架空馬のみのルールでは、以下の手順が有効：

1. **自家生産種牡馬の系統確立を狙う**
   - 血統支配率5%以上（地域）または2%以上（世界）
   - 後継種牡馬4頭以上

2. **確立時にニックス対象を操作**
   - GⅠ馬・重賞勝利馬の頭数でニックス対象系統が決定
   - 最大6種類の子系統とニックス関係成立

3. **フォースニックス達成ルート**
   - 父‐母父‐母母父‐母母母父‐母母母母で5世代のニックス関係構築

## 対戦レース攻略と必要スペック

### 凱旋門賞で勝てる馬のスペック

| 項目 | 必要値 | 備考 |
|------|--------|------|
| **スピード** | 75以上（A以上） | SP78以上で安定勝利 |
| スタミナ | 60以上 | 距離2400m対応 |
| **距離適性** | 2200-2600m含む | 2400m必須 |
| **芝質適性** | 欧州重芝向け | 日本育成馬は不利 |
| 精神力 | C+以上 | 海外遠征で能力発揮 |
| パワー | B以上 | ロンシャンの直線坂対応 |

**凱旋門賞の特徴**
- レース傾向は「持久戦」が多い（「超持久戦」「究極の持久戦」も）
- **欧州牧場で育成した馬が有利**（日本育成馬は海外レースで不利設定）
- 前哨戦としてニエル賞（9月2週）が有効

### 距離幅1600m-3000mに対応する馬の作り方

対戦レースは凱旋門賞（2400m）＋安田記念（1600m）/宝塚記念（2200m）/日本ダービー（2400m）/天皇賞秋（2000m）/菊花賞（3000m）から3レース抽選。**全レース対応には1600m-3000mの距離適性**が理想。

**柔軟性による距離適性拡張**
| 柔軟性 | 距離幅目安 |
|--------|----------|
| S | ±1000m |
| A | ±800m |
| B | ±600m |
| C | ±400m |

**距離適性を広げる方法**
1. **配合時**：柔軟性因子持ちの種牡馬・繁殖牝馬を選択
2. **調教方針**：「距離適性重視」設定
3. **史実調教**：「坂路セパレート追い」でパワー・柔軟性上昇
4. **当週調教**：「坂路調教のさきがけ」で一時的に距離適性UP

### 騎手「万能君」使用時の注意点

- **万能君**はカスタム対戦用のダミー騎手で、全条件で中立・標準的な能力
- 騎手の影響を排除した**純粋な馬の能力比較**が可能
- 特定の戦法や距離に偏りがないため、**馬自身の脚質適性**が重要
- 差し・追込馬なら瞬発力、逃げ・先行馬なら根性を重点強化
